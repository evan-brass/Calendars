Tests for the <calendar-basic>

Presentation Tests

These test show that the calendar can be styled in (hopefully) all the ways that are reasonable.

Context Tests

These tests show that the calendar can be used in a variety of different contexts and will look correct no matter how it's parents/children are styled.

Functionality Tests

These tests show that the calendar operates correctly.

Accessibility Tests

These tests show that the calendar's use of HTML, CSS, and JavaScript enable a broad audience with minimal modification by developers using the calendar.
